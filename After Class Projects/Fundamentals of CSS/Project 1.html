Introduction to CSS

CSS Stands for "Cascading Style Sheets"

It has 3 Types:-

1. Inline CSS

2. Internal CSS

3. External CSS

Inline CSS:-

Inline CSS is CSS that is applied directly to an HTML element using the "style" attribute.

Internal CSS:-

Internal CSS is CSS written within a "style" tag inside the "head" section of an HTML document. It applies styles to elements on that specific page only.
